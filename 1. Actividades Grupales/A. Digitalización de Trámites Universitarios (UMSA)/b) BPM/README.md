# BPM - Sistema de Gestión de Trámites

## Instalación y Ejecución
1. Acceder a esta carpeta desde el entorno local.
2. Ejecutar el archivo principal `/public/index.html` abriéndolo directamente en un navegador web.
3. Interactuar con la interfaz de los formularios dinámicos cargados en pantalla.
4. Registrar, consultar y actualizar datos desde los campos web; el sistema gestionará la persistencia de los datos almacenando la información de manera dinámica en archivos locales con estructura JSON, evitando la necesidad de configurar servidores de bases de datos tradicionales.
