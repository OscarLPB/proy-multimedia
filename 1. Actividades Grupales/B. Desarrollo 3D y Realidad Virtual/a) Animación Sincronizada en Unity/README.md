# Animación Sincronizada en Unity

## Instalación y Ejecución
1. Nota de almacenamiento: El paquete de desarrollo pesa `.unitypackage` (65 MB).
2. Importe este archivo directamente en su proyecto Unity3d.
3. Se lo tiene para visualizar en "/coreografia/index.html" con algun host. Lamentablemente la carpeta Build que ocupa para funcionar pesa "165 mb" lo que hace dificil subirla a GitHub, se debe importar el proyecto y hacer un Build con WebGL en Unity.
