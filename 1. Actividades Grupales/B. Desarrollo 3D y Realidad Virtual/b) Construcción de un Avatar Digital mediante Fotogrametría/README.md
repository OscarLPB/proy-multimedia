# Construcción de un Avatar Digital mediante Fotogrametría

## Instalación y Ejecución
1. Los archivos fuente y mallas tridimensionales generados mediante las herramientas de fotogrametría se encuentran organizados en este directorio.
2. Para una manipulación interactiva y visualización fluida del modelo realista (operaciones de rotación y zoom), se debe abrir el archivo "/modelo3d/index.html" con host.
