# Cover de La Vaca Lola

## Instalación y Ejecución

1. Nota de almacenamiento: El paquete de desarrollo pesado `.unitypackage` (228 MB) ha sido resguardado externamente debido a los límites de tamaño establecidos por GitHub.
2. Ingresar a esta subcarpeta desde el almacenamiento local.
3. Incluye el programa para generar un audio de la vaca lola cover con Juan Gabirel.
4. Incluye el audio con el que se entreno el modelo.
5. Incluye una imagen del proyecto en Unity.
