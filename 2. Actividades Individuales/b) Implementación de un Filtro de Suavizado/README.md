# Implementación de un Filtro de Suavizado

## Instalación y Ejecución
1. Iniciar el entorno de ejecución correspondiente al código de procesamiento digital de imágenes.
2. Ejecutar el script diseñado para el filtrado espacial.
3. Seleccionar la imagen que se desea procesar; el programa tendra varias opciones de filtro.
4. El resultado final se desplegará en la parte derecha y en la izquierda la original.
