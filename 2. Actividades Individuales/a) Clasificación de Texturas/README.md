# Clasificación de Texturas

## Instalación y Ejecución
1. Se debe importar el proyecto, en mi caso a Visual Studio 2012.
2. Tener en cuenta que esta usando SQLServer local.
3. Se pueden Cargar fotos seleccionar pixeles y guardar sus valores en rgb con alguna etiqueta como césped, tierra o cemento, etc.
4. Luego para poder Detectar diferente suelos.
